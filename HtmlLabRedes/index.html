Ultrared

Ultrared is a small network designed to emulate normal networks and their services, it has been based upon Ubuntu Server features and a small infraestructure that's going to be explain below. This network counts with 5 principal servers:

DNS: a domain name service for address properly the websites inside the network.

HTTP: a server to exchange and display hypertext markup content (web pages), this makes the network able to see all of them.

FTP: this server is for files tranfer along the network.

Streaming: a server for sharing live instant multimedia content.

SMTP/POP3: a simple mail server to mail way communication.

Security

HTTPS is the secured HTTP protocol, that implements SSL protocol to cypher the data send between host, SSL goes for Secure Socket Layer, and uses a sign certificate to check authentication of the end users in the communication. A difference between HTTP & HTTPS talking about infraestructure, is that HTTP use 80 port to communicate, thus HTTPS use 443.

Spoofing is a security communication vulnerability. It happends when an attacker injects a false DNS direction into a DNS server to redirect a user to a false website, normaly the user don't realized that this happends which make him/her give sensitive or private information to the attacker. One of the most common examples is to be redirected to a false Bank website to give your bank credentials to a hacker thats trying to steal money from your account(s).
